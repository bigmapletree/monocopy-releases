# MonoCopy

[简体中文](README_ZH.md)

Cross-platform clipboard sync tool for macOS, Windows, Linux, iOS, and Android.

Copy on one device, paste on another — text, images, rich text, and files. All communication is end-to-end encrypted (mTLS + AES-GCM). No cloud, no accounts, works entirely over your local network.

## Features

- **Real-time Clipboard Sync** — Event-driven monitoring, instant sync across paired devices
- **Multi-Content Support** — Plain text, rich text (HTML), images, and files
- **End-to-End Encryption** — mTLS for communication, X25519 key exchange, AES-GCM for data
- **Cross-Platform** — macOS, Windows, Linux, iOS, and Android
- **Clipboard History** — Searchable history with preview, source app tracking
- **File Transfer** — Chunked streaming for reliable large file transfer
- **Privacy First** — No data collection, no cloud servers, no third-party analytics
