# MonoCopy

[English](README.md)

跨平台剪贴板同步工具，支持 macOS、Windows、Linux、iOS 和 Android。

在一台设备上复制，在另一台上粘贴 — 支持文本、图片、富文本和文件。所有通信端到端加密（mTLS + AES-GCM），无需云服务、无需注册，完全在局域网内运行。

## 功能特性

- **实时剪贴板同步** — 事件驱动监控，配对设备间即时同步
- **多内容类型支持** — 纯文本、富文本（HTML）、图片和文件
- **端到端加密** — mTLS 通信加密、X25519 密钥交换、AES-GCM 数据加密
- **全平台支持** — macOS、Windows、Linux、iOS 和 Android
- **剪贴板历史** — 可搜索的历史记录，支持预览和来源应用追踪
- **文件传输** — 分块流式传输，可靠处理大文件
- **隐私优先** — 不收集数据、无云服务器、无第三方分析
